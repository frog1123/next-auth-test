# next-auth-test

nextjs website that handles users with next-auth library and prisma

## getting started

add env variables

```python
NEXTAUTH_SECRET=

// mysql
DATABASE_URL=
```

install

`pnpm i`

prisma generate

`pnpm prisma`

run dev server

`pnpm dev`

go to localhost:3000
